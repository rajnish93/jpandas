# jpandas

-   📦 Easy creation of tabular data structures in JavaScript.
-   📦 Provides a DataFrame class inspired by pandas in Python.
-   👨‍🏫 Rajnish created this CLI project

<br>

## Installation

```sh
npm install jpandas

or

yarn add jpandas
```
